# Axioz Model Router

Provides a unified interface for interacting with multiple AI model providers.  
Designed to be model-agnostic, strict-typed, and plug-and-play with the Axioz Agent Engine.

## Supported Providers

- OpenAI  
- Anthropic  
- Google Gemini  
- DeepSeek  
- Groq  
- Local: vLLM, Ollama
